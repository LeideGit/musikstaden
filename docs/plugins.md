# Recommended WordPress Plugins

Install from WP Admin → Plugins → Add New.

## Required

- **Advanced Custom Fields** (free) — band biography, media embeds, social URL fields (registered automatically by theme; ACF Pro is not required)

## Recommended

- **New User Approve** — optional extra layer; theme also checks `musikstaden_approved` user meta
- **WP Mail SMTP** — reliable email for approvals and band invites on SiteGround

## Not needed for MVP

- Page builders (Elementor, etc.)
- WooCommerce (venue payments are post-MVP)
- Caching plugins until site is live (then SiteGround Speed Optimizer is fine)
